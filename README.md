# 🐍 Snake Game

A classic Snake Game built with pure HTML5 Canvas and JavaScript — no frameworks required!

## 🎮 Features
- Smooth snake movement with grid-based physics
- Progressive difficulty: speed increases every 100 points
- Level system with score multipliers
- Particle effects on food collection
- Web Audio API sound effects
- Mobile touch & swipe support
- Local high score persistence (localStorage)
- Pause/Resume functionality

## 🚀 How to Play
1. Open `index.html` in your browser
2. Click **Start Game**
3. Use **Arrow Keys** or **WASD** to control the snake
4. On mobile: **Swipe** to change direction
5. Eat the pink food dots to grow and score points
6. Avoid walls and your own tail!
